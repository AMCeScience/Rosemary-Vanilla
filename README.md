# The awesome Rosemary

Rosemary is a flexible programming framework to build Science Gateways. Read more about Rosemary [here](http://www.ebioscience.amc.nl/gateways/rosemary/).
It has been planted as part of the [COMMIT project](http://commit-nl.nl/)!

S. Shahand and S. D. Olabarriaga, “Rosemary: A Flexible Programming Framework to Build Science Gateways,” in Proceedings of the 8th International Workshop on Science Gateways (IWSG), June 2016 , Rome, 2016. 
See [preprint](https://peerj.com/preprints/2576/)
